¿Cómo usar este editor de código Python?

Este editor de código Python te permite escribir y ejecutar código Python directamente en tu navegador. Para usarlo, sigue estos pasos:

  1. Escribe tu código Python en el área de texto proporcionada.
  2. Haz clic en el botón "Evaluá tu código" para ejecutar el código que has escrito.
  3. El resultado de la ejecución se mostrará debajo del botón, en la sección "Resultado".

Este editor admite caracteres comprendidos en el conjunto de caracteres UTF-8. Por ejemplo:

  • Caracteres acentuados: á, é, í, ó, ú
  • Caracteres especiales: ñ, ü, ç
  • Símbolos: @, #, $, %, ^, &, *, (, )
  • Comillas rectas: ', ". NO comillas curvas: “, ”

Puedes escribir cualquier código Python válido, desde operaciones simples hasta funciones más complejas. ¡Diviértete programando!
